# 코드 동작 페이지

이 페이지는 "IT Cookbook, HTML5 프로그래밍 입문 (3판)"에서 설명하는 샘플 코드와 동작 데모를 제공하는 페이지입니다. 긴 코드를 복사해서 붙여넣기거나, 동작 데모를 확인하기 위한 용도이므로, 코드의 내용과 기능은 책을 읽어 주시면 감사하겠습니다.

모든 파일이 이 사이트에 있는 것은 아닙니다

기본 예제 위주로 정리되어 있습니다. 참고를 위한 일부 코드는 한빛아카데미 사이트에서 다운로드 해주세요.

# 자바스크립트 부분과 관련된 이야기

자바스크립트 코드 부분의 alert() 함수를 책과 달리 document.write() 메서드로 대체했습니다. 이는 페이지에 들어갔을 때 경고창이 무수하게 뜨는 것을 막기 위함으로, 단순하게 출력을 확인하기 위한 용도로 사용한 alert() 함수와 큰 차이는 없습니다.
